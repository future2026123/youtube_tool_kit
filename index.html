상품별 안내에 따라 다운로드, 설치, 라이선스 정보를 확인할 수 있습니다. 맞춤 제작이 필요하다면 문의 내용을 바탕으로 제작 범위와 일정을 먼저 정리합니다.

  1. 도구 구매필요한 프로그램을 선택하고 구매 페이지에서 결제를 진행합니다.
  2. 사용 안내상품별 방식에 따라 다운로드, 설치, 라이선스 정보를 확인합니다.
  3. 기능 의뢰원하는 기능이 있다면 사용 환경과 필요한 작업을 정리해 문의합니다.
  4. 범위 확인문의 내용을 바탕으로 기능 범위, 예상 일정, 견적 기준을 안내받습니다.

FAQ

자주 묻는 질문

구매하면 프로그램은 어떻게 받나요?

결제 완료 후 상품별 안내 방식에 따라 다운로드 링크, 설치 파일, 또는 사용 안내를 받게 됩니다.

어떤 결제 수단을 사용할 수 있나요?

각 제품의 구매 버튼을 누르면 해당 상품의 결제 페이지로 이동합니다. 상품에 따라 제공되는 결제 수단이 다를 수 있습니다.

라이선스는 몇 대의 PC에서 사용할 수 있나요?

라이선스 범위는 상품별 안내를 따릅니다. 구매 전 사용 인원, 설치 가능 PC 수, 팀 사용 가능 여부를 확인해 주세요.

Windows와 macOS에서 사용할 수 있나요?

지원 운영체제는 프로그램마다 다를 수 있습니다. 구매 전 Windows, macOS 지원 여부와 최소 버전을 확인해 주세요.

구매 후 업데이트를 받을 수 있나요?

업데이트 제공 여부는 상품별로 다릅니다. 업데이트가 제공되는 경우 구매자 이메일 또는 안내 페이지를 통해 공지합니다.

구매 후 환불이 가능한가요?

디지털 상품 특성상 다운로드 또는 라이선스 발급 후에는 환불이 제한될 수 있습니다. 구매 전 상품별 환불 조건을 확인해 주세요.

설치나 실행 중 문제가 생기면 어떻게 하나요?

구매 이메일, 운영체제, 오류 화면, 프로그램 이름을 함께 보내주시면 확인에 도움이 됩니다.

팀이나 회사에서 여러 명이 사용해도 되나요?

여러 명이 사용할 계획이라면 구매 전 문의해 주세요. 사용 인원과 목적에 맞는 라이선스 또는 별도 제작 범위를 안내할 수 있습니다.

원하는 기능으로 따로 제작 의뢰할 수 있나요?

가능합니다. 필요한 기능, 사용 환경, 참고 자료, 원하는 납기일을 정리해 보내주시면 제작 범위와 견적을 안내합니다.

커스텀 제작 비용은 어떻게 정해지나요?

기능 수, 화면 구성, 자동화 난이도, 데이터 저장 방식, 유지보수 필요 여부에 따라 달라집니다. 먼저 필요한 기능을 기준으로 범위를 정리합니다.

새 프로그램이 추가되면 어디서 확인하나요?

새로운 프로그램은 프로그램 섹션에 순차적으로 추가됩니다. 업데이트나 새 상품 안내는 페이지와 문의 채널을 통해 확인할 수 있습니다.
